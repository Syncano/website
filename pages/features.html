Real-Time Sync

Communication API

Syncano’s synchronization tools ensure your data flows seamlessly across devices and servers in real-time. You can use the following three transport methods:

  • XML-RPC
  • JSON-RPC
  • RESTful API

Sync Server

Our Sync Server provides real time notification/subscription processing across all of your apps. Connect via:

  • WebSocket
  • TCP

Complex Modeling

Hierarchies

Organize your data using our Projects, Collections, and Folders hierarchies and create any data structure with many-to-many relations.

Data Storage

Sync and store your data as Data Objects in Syncano. Our platform scales with your needs, so you don’t have to worry about sudden bursts in usage.

More Than Storage

When storing images, Syncano gives you additional tools such as automatic thumbnail generation.

Admin GUI

A Simpler Way to Manage Your Apps

Syncano’s GUI enables you and your team to manage your apps quickly and efficiently:

  • Visually organize and manage your Data Objects, users, and administrators without a single line of code
  • Preview and moderate your content

Access Management

Manage who can access your data and differentiate user experiences.

Roles

Give team members different levels of access to your API and Admin GUI.

Multiple Instances

Set up several different instances under one account with independent access URLs.

User Authorization

Restrict user access to specific Projects, Collections, Folders and Data Objects.
